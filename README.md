# Steam游戏拼图生成器

根据Steam游戏时长生成个性化拼图,游戏时长越长,封面越大!

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 本地运行
```bash
npm run dev
```
访问 http://localhost:3000

### 3. 构建生产版本
```bash
npm run build
npm start
```

## 📦 部署到Vercel

1. 将代码推送到GitHub
2. 访问 [Vercel](https://vercel.com)
3. 导入你的GitHub仓库
4. 点击Deploy
5. 完成!

## 🎮 使用方法

1. 获取Steam ID: https://steamid.io/
2. 获取API密钥: https://steamcommunity.com/dev/apikey
3. 确保Steam资料设置为公开
4. 输入信息并生成拼图
5. 下载你的专属游戏拼图!

## ✨ 功能特性

- ✅ 动态调整封面大小(基于游戏时长)
- ✅ 自动优化输出分辨率
- ✅ 显示所有游玩过的游戏
- ✅ 无缝矩形布局
- ✅ 一键下载高清PNG
- ✅ 完全客户端运行,保护隐私
